 # OptiCrop: Smart Agricultural Production Optimization Engine

## Project Description
OptiCrop is a machine learning-based web application that recommends the best crop based on soil nutrients and environmental conditions.

## Features
- Crop Recommendation
- Soil Analysis
- Environmental Suitability
- Machine Learning Prediction

## Technologies Used
- Python
- Flask
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- SciPy
- HTML
- CSS

## Dataset
Crop Recommendation Dataset
